# Smart Bridge - Personalized Networking Assistant

## Overview
Smart Bridge is an AI-powered networking assistant built using Streamlit and Google's Gemini API. It generates personalized conversation starters and networking tips based on a user's profile and event details.

## Features
- AI-generated conversation starters
- Personalized networking tips
- User profile customization
- Wikipedia topic search
- Simple Streamlit interface

## Technologies Used
- Python
- Streamlit
- Google Gemini API
- Wikipedia API
